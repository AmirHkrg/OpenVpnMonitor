# OpenVpnMonitor
OpenVPN server monitoring

## Install 
### Step 1
###### Windows
Open `requirement.bat` file
###### Linux/Mac
```
pip install -r requirement.txt
```
### Step 2
Place the file `OpenVpnMonitor.py` next to the file `openvpn-status.log`.

### Step 3
```
python OpenVpnMonitor.py
```
